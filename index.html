Sincronización en tiempo real
Conecta con Google Sheets para ver datos en celular y laptop
✅ Cómo funciona:
1. Pegas la URL una sola vez y haces clic en "Conectar"
2. El app se descarga automáticamente con la URL incluida
3. Tienda 1 y Tienda 2 se sincronizan automáticamente
100% gratis · Sin tarjeta · Solo tu Gmail
⚠ MUY IMPORTANTE:
NO edites los datos directamente en Google Sheets.
Google Sheets es solo un puente entre dispositivos — editarlo directamente puede dañar los datos.
Siempre agrega, edita o elimina productos desde el app.
Si algo quedó mal en Sheets, usa el botón 🔧 Reparar Sheets para restaurarlo.
📋 Instrucciones para conectar (5 minutos)
1. Ve a script.google.com con tu Gmail
2. Clic en "Nuevo proyecto"
3. Borra todo el código y pega el código del archivo chikarella_script.gs
4. Clic en Implementar → Nueva implementación
5. Tipo: Aplicación web · Acceso: Cualquier persona
6. Copia la URL que aparece y pégala arriba
7. Abre el mismo archivo HTML en el celular y la laptop — ¡ya se sincronizan!
💾
Backup y restauración de datos
Exporta todos tus datos para no perderlos al actualizar el app
💡 Importante: Cuando descargues una nueva versión del app, primero exporta un backup aquí.
Luego abre la nueva versión y usa "Importar backup" — recuperas todos tus productos, ventas y la URL de Google Sheets sin tener que configurar nada de nuevo.
⚠ Zona de peligro
📲 Instalar como app + Escáner con cámara
Para instalar en celular y usar el escáner de cámara:
1. Sube el archivo a netlify.com (gratis, arrastra el archivo HTML)
2. Te da un link como https://xxxxx.netlify.app
3. Abre ese link en Chrome del celular
4. Chrome muestra "Instalar app" o usa menú ⋮ → Instalar
5. ¡Ya tienes la app instalada con ícono en el celular! 📱
Si ya está instalada: abre desde el ícono en tu pantalla de inicio.
